# Textbooks — acquisition checklist

Download into this folder using the suggested filenames so the in-notebook
`file:///.../textbooks/<name>.pdf` links resolve. ⭐ = primary anchor (get these first).

## Free & legal — download directly

- ⭐ **MacKay.pdf** — Information Theory, Inference, and Learning Algorithms (MacKay)
  https://www.inference.org.uk/itprnn/book.pdf  *(author release; on-screen viewing licensed)*
- ⭐ **Stone.pdf** — Information Theory: A Tutorial Introduction (Stone)
  https://arxiv.org/pdf/1802.05968  *(Creative Commons)*
- ⭐ **PolyanskiyWu.pdf** — Information Theory: From Coding to Learning (Polyanskiy & Wu)
  https://people.lids.mit.edu/yp/homepage/data/itbook-export.pdf  *(free pre-pub draft, personal use)*
- **RichardsonUrbanke.pdf** — Modern Coding Theory (Richardson & Urbanke)
  https://documents.epfl.ch/groups/i/ip/ipg/www/2010-2011/Statistical_Physics_for_Communication_and_Computer_Science/mct-new.pdf
- **Grunwald-MDL.pdf** — A Tutorial Introduction to the MDL Principle (Grünwald)
  https://arxiv.org/pdf/math/0406077
- **Yeung.pdf** — Information Theory and Network Coding (Yeung)
  https://iest2.ie.cuhk.edu.hk/~whyeung/post/draft2.pdf  *(author draft)*
- **Moser-notes.pdf** — free ETH lecture notes substitute for Moser & Chen
  https://moser-isi.ethz.ch/scripts.html
- **Gallager-LDPC.pdf** — Gallager's 1963 LDPC monograph (free), as a Gallager stand-in
  https://www.webee.technion.ac.il/people/sason/Gallager_monograph.pdf
