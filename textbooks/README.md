# References - public link registry

The GitHub Pages site links to public book pages, author PDFs, arXiv papers,
OpenReview entries, and library/publisher records. No PDFs are committed to
this repository, and the exported notebooks should not contain machine-local
filesystem links.

  https://www.inference.org.uk/itprnn/book.pdf
- **Stone.pdf** - Information Theory: A Tutorial Introduction (Stone)
  https://arxiv.org/pdf/1802.05968
- **PolyanskiyWu.pdf** - Information Theory: From Coding to Learning (Polyanskiy and Wu)
  https://people.lids.mit.edu/yp/homepage/data/itbook-export.pdf
